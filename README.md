Full-Stack Web Application

Designed and developed a full-stack workout tracking platform for logging workouts, scheduling sessions, and visualizing long-term fitness progress.</br>
Implemented secure user authentication and authorization with JWT, enforcing protected routes and role-based access.</br>
Built RESTful backend services with Python, FastAPI, and MongoDB, including complex aggregation queries for weekly/monthly progress reports.</br>
Developed responsive frontend views with TypeScript, React, and Tailwind CSS, integrating calendars, scheduling workflows, and real-time UI updates via state management.</br>
